Nội dung từ Load Data Target

Nội dung này được tải khi bạn cố gắng gửi dữ liệu POST. Trong một ứng dụng web thực, file này sẽ là một script (ví dụ: PHP, Node.js, Python) nhận dữ liệu POST và tạo ra phản hồi động dựa trên dữ liệu đó.

Hiện tại, vì đây là HTML tĩnh, nó không thể hiển thị dữ liệu bạn đã gửi.

Tuy nhiên, nếu bạn mở tab Network (F12) trong trình duyệt, bạn sẽ thấy dữ liệu POST đã được gửi đi!
